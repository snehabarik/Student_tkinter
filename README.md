# 🎵 Musify Login GUI

A simple login interface built using **Python Tkinter** and **PIL**.

## ✨ Features

* Clean and styled GUI design
* Email and Password input fields
* Login button with validation
* Image and custom icon support
* Accepts any email/password (for demo purposes)

## 🛠️ Technologies Used

* Python
* Tkinter
* PIL (Pillow)

## 🚀 How to Run

1. Install required library:

   ```
   pip install pillow
   ```
2. Run the Python file:

   ```
   python your_file_name.py
   ```

## 📌 Note

* This is a basic demo project.
* It does not use a database or real authentication system.
